## :gear: Setup

```bash
# Clone repo
$ git clone https://github.com/ryabykhms/rsclone-backend

$ cd rsclone-backend

# Switch to the develop branch
$ git checkout develop

# Install all dependencies
$ npm install

# Run
$ npm run dev

# Go to http://localhost:5000/ on your web browser.
```
